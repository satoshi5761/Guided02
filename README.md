# Belanja Diskon

## Deskripsi
Selain menjadi guru matematika, Wisnu juga seorang pedagang. Saat ini, toko Wisnu sedang mengadakan diskon dalam bentuk persentase (%).  
Karena banyaknya pesanan yang masuk, Wisnu kesulitan menghitung total belanjaan para pelanggan.

Oleh karena itu, dibuatlah sebuah program untuk menghitung total belanja pelanggan setelah diskon.

---

## Ketentuan Program
- Program **wajib menggunakan fungsi**
- Hasil perhitungan harus **dibulatkan hingga 2 angka di belakang koma**
- Gunakan fungsi pembulatan (misalnya `round()`)

---

## Spesifikasi Fungsi
Nama fungsi:
```python
belanja(harga, jumlah, diskon)


## Contoh

| Test | Result |
|------|--------|
| `print(belanja(100000, 5, 10))` | `450000.0` |
| `print(belanja(1570000, 8, 55))` | `5652000.0` |
